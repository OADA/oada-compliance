# OADA Conformance Tests #
Automated tests for OADA API


## Running all tests ##
    npm run check

## Specifying a config file ##
    CONFIG='/path/to/file.json' npm run check

## Can pass mocha options (e.g.) ##
    npm run check -- --grep well-known
